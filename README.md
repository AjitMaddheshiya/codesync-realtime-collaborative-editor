# 🧑‍💻 CodeSync - Real-Time Collaborative Code Editor

**CodeSync** is a real-time collaborative code editor built with **Node.js**, enabling multiple users to write, edit, and view code together live. It features both client and server-side logic to facilitate real-time syncing of code using **WebSockets**.

---

## 🚀 Features

- 🧠 Real-time collaborative editing
- 🧑‍🤝‍🧑 Multiple users can join the same session
- 🌐 Client-Server architecture
- 🧾 Syntax highlighting (optional with extensions)
- 🔒 Unique session rooms (if implemented)
- ⚡ Fast and responsive editing using **Socket.IO**
